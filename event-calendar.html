WordPress イベントカレンダーでは、各地の WordBench 勉強会やオープンソース・PHP 関連のカンファレンス、セミナーなど、WordPress ユーザーの皆さんに興味を持っていただけるようなイベントを紹介しています。

各種フォーマット

  • XML
  • iCal

掲載イベント

このカレンダーには以下のようなイベント情報を掲載しています。

  • WordBench イベント
  • それ以外のコミュニティ運営イベント

「コミュニティ運営イベント」とは、特定の企業・団体や個人などの宣伝や利益を上げる目的ではないイベントを指します。有料のイベントも掲載可能ですが、その場合は会場費など実費のみを徴収することが条件となります。

いずれの場合も、WordPress の商標の取り扱いやライセンスへの正しい理解を促進するようなイベントである必要があります。

イベントの追加
追加したいイベントがある場合は、分かる範囲内で以下の情報を添えてお問い合わせフォームからご連絡ください。件名は「イベントカレンダー」としてください。

  • イベントの名称
  • 日時（開始・終了時間）
  • 場所
  • 関連 URL
  • 簡単な説明（数行程度。詳細は関連 URL のリンク先にご記入ください）
